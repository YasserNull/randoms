# Physics Addon

![Downloads](https://img.shields.io/github/downloads/YasserNull/Physics-Addon/total?label=Downloads)
![Stars](https://img.shields.io/github/stars/YasserNull/Physics-Addon?style=social)

**Physics Addon** adds stunning physics effects to the Minecraft world, bringing a more dynamic and immersive experience.

---

## Features

### Block Breaking
When a player breaks a block, small fragments appear and then disappear.

![Block Breaking](https://i.imgur.com/AeoxLaF.png)

### Block Explosions
When blocks explode, block fragments are physically propelled with motion.

![Block Explosions](https://i.imgur.com/B31SxiS.png)

### Physics Addon Settings
This lets you configure limits for explosion mass and physics behavior.

![Addon Settings](https://i.imgur.com/ckV3I5x.jpeg)

---

## Installation

Download the latest version from the link below and import it into Minecraft:

- [**Download Latest Release**](https://github.com/YasserNull/Physics-Addon/releases)

> **Note:** Make sure you are using a compatible version of Minecraft. This addon may affect performance on low-end devices due to heavy physics effects.
